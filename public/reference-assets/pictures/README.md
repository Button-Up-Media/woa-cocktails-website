# WOA picture library

This is the photo-first asset library for the WOA rebuild.

Primary folders:
- `home`
- `about`
- `classes`
- `contact`
- `events/index`
- `events/style-bender`
- `events/summer-party`
- `events/ink-cocktail-class`

Use the pictures here first when building homepage, service pages, gallery sections, and location pages.
Brand graphics and icons stay in `public/reference-assets/current-site/brand` as secondary support assets.
